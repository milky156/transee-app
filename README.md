# Transee Butuan

This is the Public Transport Navigation App for Butuan City.

## Features

- **Route Finder**: Find the best multicab route to your destination.
- **Fare Estimator**: Calculate your commute fare based on city ordinances.
- **Transport Map**: Interactive map with real-time route visualization.

## Running the code

1. Run `npm i` to install the dependencies.
2. Run `npm run dev` to start the development server.
